# Connect Four

## Description
This is a standalone web application for connect 4. It is built for two local players, and does not have any AI to play against.


## Features
 - Two Player Local Gameplay
 - Automatic Turn Switching
 - Win/Tie Detection
 - Alternates Which Player Goes First After Ever Round
 - Non-Persistent Score Tracking

## How To Run
1. Download As Zip Or Clone Repository
2. Unzip/Open The Project Folder
3. Double Click index.html Or Open In Your Browser Manually

## Controls
 - Click The Buttons Over The Columns Or Press 1-7 On Your Keyboard
 - Press R To Reset The Board, Or H To Open/Close The "How To Play" Menu

## Built Using:
 - HTML
